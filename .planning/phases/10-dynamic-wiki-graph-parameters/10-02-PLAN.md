ForceGraph2D d3Force API reminders (graphRef.current methods):
- `g.d3Force('link').distance(fn)` — set per-link distance; applies on next tick
- `g.d3Force('link').strength(n)` — link spring strength
- `g.d3Force('center').strength(n)` — pull toward center
- `g.d3Force('charge').strength(n)` — NEGATIVE values repel, positive attract
- `g.d3ReheatSimulation()` — re-heats simulation (alpha → 1) — AVOID during slider drag
- `g.d3Force('charge').alphaTarget(0.1)` — keep simulation warm at target alpha
- Radix: `<Slider.Root onValueChange onValueCommit onPointerDown onPointerUp>`; pass `[value]` as array.

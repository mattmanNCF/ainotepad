**Test 3 — Serial queue (burst submission)**

1. Force-quit the app while a note is in pending state (submit note, immediately close before ✓).
2. Relaunch with `npm run dev`.
3. The previously pending note should eventually transition to ✓.

Expected: reQueuePendingNotes() picks up the note on launch.
